## A Banner for promotional content

Phone screen variant to allow for quick phone mockups. Allow for most parts to be replaced by custom components.

### Component usage

Modify the text to see it change live:

```js live
<Banner
  title="On The Journey"
  description="Know Jesus Today!"
  backgroundSrc="https://source.unsplash.com/random/1920x1080"
  footer="For everyone"
  py={20}
/>
```
