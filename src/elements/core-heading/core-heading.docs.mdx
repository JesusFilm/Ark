import { CoreHeading } from './core-heading';

## Render Headings

Render an h2 heading

### Component usage
```js
<CoreHeading {...{
    name: 'core/heading',
    attributes: {
      align: 'left',
      content: 'The parables from Matthew',
      level: 2
    }
  }} />
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<CoreHeading {...{
    name: 'core/heading',
    attributes: {
      align: 'left',
      content: 'The parables from Matthew',
      level: 2
    }
  }} />
```
