## React component that displays the author card

A component that display a single author with name, description and profile image. 

### Using props to customize the text

Modify the props to see it change live
```js live
<AuthorCard
  author={{
    node: {
      name: 'Tez Brooks',
      slug: 'tez-brooks',
      avatar: {
        url: ''
      },
      description:
        'Award winning author, screenwriter and international speaker'
    }
  }}
/>
```
