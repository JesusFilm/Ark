import { TimeAgo } from './time-ago';

TimeAgo example:

Accepts a date object and optional locale and returns a human readable time ago string.

Code Snippet:
```js
const d = new Date()
d.setDate(d.getDate() - 5)
<TimeAgo datetime={d} locale='ru' />
```


Live Playground:
```js live
<TimeAgo datetime={new Date()} locale='enUS' />
```
